重点笔记
TCP 三次握手为什么是三次？防止已失效的连接请求报文段突然又传送到了服务端。
TCP 四次挥手为什么是四次？TCP 是全双工的，双方都要各自关闭连接。

TCP vs UDP
| 特性 | TCP | UDP |
|------|-----|-----|
| 连接性 | 面向连接 | 无连接 |
| 可靠性 | 可靠 | 不可靠 |
| 速度 | 慢 | 快 |
| 头部大小 | 20字节 | 8字节 |
| 应用场景 | HTTP、FTP、SSH | DNS、视频、语音 |

常用命令记录
文件操作
| 命令 | 作用 |
|------|------|
| `ls` | 列出文件 |
| `ls -l` | 详细列表 |
| `ls -la` | 含隐藏文件 |
| `cd 目录` | 切换目录 |
| `cd ..` | 返回上一级 |
| `pwd` | 显示当前路径 |
| `mkdir 目录` | 创建目录 |
| `mkdir -p a/b/c` | 递归创建多级目录 |
| `touch 文件` | 创建空文件 |
| `rm 文件` | 删除文件 |
| `rm -rf 目录` | 递归删除目录 |
| `cp 源 目标` | 复制 |
| `mv 源 目标` | 移动/重命名 |
| `cat 文件` | 查看文件内容 |

 系统查看
| 命令 | 作用 |
|------|------|
| `uname -a` | 内核版本 |
| `cat /etc/redhat-release` | 系统版本 |
| `hostname` | 主机名 |
| `free -h` | 查看内存 |
| `df -h` | 查看磁盘 |
| `top` | 实时监控（按q退出） |
| `ps aux` | 查看所有进程 |
| `ps aux | grep nginx` | 查找指定进程 |
| `netstat -tulnp` | 查看端口监听 |
| `ip addr` | 网卡和IP信息 |
| `systemctl status 服务` | 查看服务状态 |

 vi 编辑器
- `i`：进入插入模式
- `Esc`：回到命令模式
- `:w`：保存
- `:q`：退出
- `:wq`：保存并退出
- `:q!`：不保存强制退出
- `dd`：删除整行
- `yy`：复制整行
- `p`：粘贴
- `/关键字`：搜索
